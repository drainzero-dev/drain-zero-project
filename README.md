# Drain Zero

Drain Zero is a professional tax optimization web application designed to help users minimize tax leakage and make smart fiscal decisions.

## 🚀 Tech Stack
- **Frontend:** React 19 (Vite)
- **UI Framework:** Ant Design (antd)
- **Routing:** React Router DOM
- **Design:** Modern Fintech Style

## ✨ Features
- **Landing Page:** Professional hero section with FY 2025-26 readiness badge.
- **Login Page:** Clean authentication interface with Google integration support.
- **Signup Page:** Streamlined onboarding for new users.

## 📁 Folder Structure
```text
src/
  assets/          # Images and Icons
    images/
    icons/
  components/      # Reusable UI components
    common/        # Shared components (Buttons, Inputs, etc.)
    auth/          # Auth-specific components
  pages/           # Page containers
    Landing/       # Landing page source
    Auth/          # Login and Signup pages
  routes/          # Application routing configuration
  styles/          # Global styles and theme tokens
  constants.js     # Application-wide constants
  App.jsx          # Root application component
  index.js         # Entry point
```

## 🛠️ How to Run
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
